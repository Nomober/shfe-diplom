## Удаление сеанса

## ![DELETE](../../img/delete.svg) / seance/{seanceId}

**_Удаляет_** сеанс из системы.  
**_Возвращает_** список всех сеансов

## Параметры:

- **seanceId** - ID удаляемого сеанса  (Например `34`) - указывают в url адресе

## Результат (`result`)

```javascript  
{  
  seances: [], // Список сеансов 
}  
```
